# Project Concept

## 한 줄 소개

> 별도의 가입 없이, 비밀번호가 설정된 고유 링크를 통해 안전하게 문서를 공유함.

## 타겟 유저 & 핵심 시나리오

**타겟 유저**
- 민감한 정보를 특정 개인에게 일회성으로 전달하고자 하는 사용자.

**핵심 시나리오**
1. **작성**: 메인 페이지 접속 후 WYSIWYG 에디터로 본문 작성 및 비밀번호 설정.
2. **생성**: 저장 시 서버에서 고유 URL 반환.
3. **공유**: 작성자가 수신자에게 해당 URL 전달.
4. **인증**: 수신자가 URL 접속 시 비밀번호 입력 폼 노출.
5. **조회**: 비밀번호 일치 시 본문 데이터를 화면에 렌더링.

## 핵심 기능

**MVP**
- [ ] 이미지 업로드, 텍스트 스타일링이 가능한 WYSIWYG 인터페이스.
- [ ] 게시글 저장 시 비밀번호 필수 입력.
- [ ] 추측 불가능한 상세 페이지 URL.
- [ ] 유효한 비밀번호 입력 전까지 본문 데이터 비공개.
- [ ] 링크 생성 완료 후 원클릭 URL 복사 기능.

**나중에**
- [ ] 댓글
- [ ] 공개 게시글
- [ ] 유료 플랜 (조회 수 증가에 따른 서버 비용 대응)
